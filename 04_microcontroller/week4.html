<br> <h1>Microcontroller Programming </h1> Our task was to use a microcontroller to program something. I decided to program my microcontroller, so that when a button is pressed the motor will turn on or off. The idea was that pressing the motor will bring the motor to a compelteboard stop. However, I ended up inadvertently creating a motion sensor. This was the result of me forgetting to add a resistor to my breadboard. Whenver you get remotely close to the button or barely even touch it, the motor will activate. Personally, I think this is pretty cool, so I decided to leave the project the way it is. <h4>Pseudocode</h4> Start Program <br>Set motor as output</br> Set button as input <br>Make sure motors are off</br> Infinite Loop <br>Is the button pressed?</br> Yes → Turn on motor <br>No → Turn off motor</br> <br></br> <h4>Code</h4> <pre style="border:2px solid #BCBEC0; padding: 10px; font: 14px Monaco; width:70%; background: #f7f7f9;border-radius: 25px;"><code> const int buttonPin = 2; // the number of the pushbutton pin const int A1A = 3; // define pin 3 for A-1A const int A1B = 4; // define pin 4 for A-1B int buttonState = 0; // variable for reading the pushbutton status - will change void setup() { pinMode(buttonPin, INPUT); // initialize the pushbutton pin as an input: pinMode(A1A, OUTPUT); // specify these pins as outputs pinMode(A1B, OUTPUT); digitalWrite(A1A, LOW); // start with the motors off digitalWrite(A1B, LOW); } void loop() { // read the state of the pushbutton value buttonState = digitalRead(buttonPin); // check if the pushbutton is pressed. If it is, the buttonState is HIGH: if (buttonState == HIGH) { // turn Motor on: digitalWrite(A1A, HIGH); digitalWrite(A1B, LOW); } else { // turn Motor off: digitalWrite(A1A, LOW); // setting both pins LOW stops the motor digitalWrite(A1B, LOW); // redundant, but doesn't hurt } }</code> </pre> <h4>Wiring</h4> <img src="https://lh4.googleusercontent.com/bRv2-MHhvdgjX9q8kyHCaueSoumqxAk63TYHbHrHuMxWOs7ZkWvC45IREL-gBjLVpT49GltDHDQkWy66Sj4ZK7_c1XLsN-9RXxgj_tSmOLpeFIGOvUi87WKnOGsUEk1K6-YSrCLNMLsm8kPzbWAaieA" alt="Image"> <p></p> <img src="https://lh5.googleusercontent.com/hDQZWXSweeRki7KqhGZutDjZS4UVhlfhV9FWdNdxIPcNNsn6poa7zzuOosnHZ0SY1UOGMwYLbXwGq-JP3Klyq0UB22GycoxORoA0Jx6Bh5FVgu4yyFpOSmNO1VSWwdowl0MbKIYx79-yNLnrY2y-4fM" alt="Image"> <p></p> <img src="https://lh3.googleusercontent.com/VWlSJ1fPiZqZOWIUsKf-3PyOtk1yYfmhs4EsOwL5iyb4u2u7U1S8YRYICBnzjzCesJ-Ckly0SMcF5BbSEnbb9msAiBt7SzUFS0BLWFA9g0g7bmT0x3jzvjnVhkOtfZVg1tvdebKkJRD08Lrw2gQB2m4" alt="Image"> <p></p> <h4>Circuit Diagram</h4>
